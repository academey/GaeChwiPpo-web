## 기여하기

개취뽀 웹사이트는 오픈 소스 프로젝트이며, 누구나 자유롭게 기여할 수 있습니다. 다음과 같은 방법으로 기여하실 수 있습니다:

1. 이슈 등록
   - 버그 리포트
   - 새로운 기능 제안
   - 문서 개선 제안

2. Pull Request
   - 코드 개선
   - 새로운 기능 추가
   - 버그 수정

3. 문서 작성
   - README 개선
   - 코드 주석 추가
   - 사용 가이드 작성

## 프로젝트 구조

이 프로젝트는 [v0.dev](https://v0.dev)를 사용하여 구축되었으며, Vercel을 통해 자동으로 배포됩니다.

## 개발 환경 설정

1. 저장소 클론
```bash
git clone https://github.com/your-username/gaechwippo-web.git
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm run dev
```
